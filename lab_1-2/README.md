# Лабораторные работы №1-2
## Современные технологии ООП

Этот проект содержит реализацию двух лабораторных работ по объектно-ориентированному программированию.

## Структура проекта

```
lab_1-2/
├── docs/                           # Документация
│   ├── lab_1.txt                  # Задание к лабораторной работе 1
│   ├── lab_2.txt                  # Задание к лабораторной работе 2
│   ├── Пояснения_к_коду.md        # Пояснения к коду
│   └── СТООП_лаб1_2.pdf          # Методические указания
├── lab1-rational-numbers/          # Лабораторная работа 1
│   ├── src/
│   │   ├── RationalNumber.cs      # Класс рациональных чисел
│   │   └── Program.cs             # Демонстрационная программа
│   └── RationalNumberProject.csproj
├── lab2-complex-numbers/           # Лабораторная работа 2
│   ├── src/
│   │   ├── ComplexNumber.cs       # Класс комплексных чисел
│   │   └── Program.cs             # Демонстрационная программа
│   └── ComplexNumberProject.csproj
└── README.md                      # Этот файл
```

## Лабораторные работы

### Лабораторная работа №1: Класс рациональных чисел
- Реализация класса для работы с обыкновенными дробями
- Арифметические операции, сравнения
- Парсинг строк формата "4:5" и "4/5"
- Автоматическое сокращение дробей
- Обработка исключений

### Лабораторная работа №2: Класс комплексных чисел
- Реализация класса для работы с комплексными числами
- Арифметические операции
- Парсинг строк формата "2+4i", "-4i"
- Специальные элементы (ноль, единица, обратный)
- Обработка исключений

## Запуск проектов

### Лабораторная работа 1:
```bash
cd lab1-rational-numbers
dotnet run
```

### Лабораторная работа 2:
```bash
cd lab2-complex-numbers
dotnet run
```
