# Лабораторная работа №1: Класс рациональных чисел

## Описание
Реализация класса для работы с рациональными числами (обыкновенными дробями) с поддержкой всех основных математических операций.

## Возможности
- **Конструкторы**: создание дроби с двумя параметрами, одним параметром или без параметров
- **Парсинг строк**: поддержка форматов "4:5" и "4/5"
- **Автоматическое сокращение**: дроби приводятся к несократимому виду
- **Арифметические операции**: +, -, *, /
- **Операции сравнения**: ==, !=, <, >, <=, >=
- **Специальные элементы**: Zero, One, Inverse
- **Генерация случайных чисел** на заданном интервале
- **Обработка исключений**: деление на ноль, неверный формат

## Файлы
- `src/RationalNumber.cs` - основной класс
- `src/Program.cs` - демонстрационная программа
- `RationalNumberProject.csproj` - файл проекта

## Запуск
```bash
dotnet run
```

## Примеры использования
```csharp
var r1 = new RationalNumber(4, 8);  // 1/2 (автоматически сокращается)
var r2 = RationalNumber.Parse("6/9");  // 2/3
var result = r1 + r2;  // 7/6
```
