# Лабораторная работа №2: Класс комплексных чисел

## Описание
Реализация класса для работы с комплексными числами с поддержкой всех основных математических операций.

## Возможности
- **Конструкторы**: создание комплексного числа с двумя параметрами, одним (вещественная часть) или без параметров
- **Парсинг строк**: поддержка форматов "2+4i", "-4i", "3.5-2.1i", "7"
- **Арифметические операции**: +, -, *, /
- **Операция равенства**: == и !=
- **Специальные элементы**: Zero, One, Inverse
- **Генерация случайных чисел**
- **Обработка исключений**: деление на ноль, неверный формат

## Файлы
- `src/ComplexNumber.cs` - основной класс
- `src/Program.cs` - демонстрационная программа
- `ComplexNumberProject.csproj` - файл проекта

## Запуск
```bash
dotnet run
```

## Примеры использования
```csharp
var c1 = new ComplexNumber(2, 3);  // 2+3i
var c2 = ComplexNumber.Parse("1-2i");  // 1-2i
var result = c1 * c2;  // 8-i
```

## Математические формулы
- **Сложение**: (a+bi) + (c+di) = (a+c) + (b+d)i
- **Умножение**: (a+bi) × (c+di) = (ac-bd) + (ad+bc)i
- **Деление**: (a+bi) / (c+di) = [(ac+bd) + (bc-ad)i] / (c²+d²)
