# Современные технологии ООП

## Обзор

Репозиторий содержит лабораторные работы по курсу "Современные технологии объектно-ориентированного программирования" на C# (.NET 8.0).

```
oop-c-/
├── lab_1-2/                    # Лабораторные 1-2 (сданы)
│   ├── lab1-rational-numbers/  # Рациональные числа
│   └── lab2-complex-numbers/   # Комплексные числа
└── lab_3-5/                    # Лабораторные 3-5
    ├── lab3-field-interface/   # Интерфейс IField<T>
    ├── lab3-rational-numbers/  # RationalNumber : IField<T>
    ├── lab3-complex-numbers/   # ComplexNumber : IField<T>
```

## Лабораторная работа №3: Интерфейс математического поля

**Суть**: Реализация интерфейса `IField<T>` для работы с алгебраическими полями (рациональные и комплексные числа).

**Основные компоненты**:
- `IField<T>` - интерфейс поля с операциями (+, -, *, /, ==, !=)
- `RationalNumber` - рациональные числа (точная арифметика)
- `ComplexNumber` - комплексные числа
- `Vector<T>` - векторы над полем
- `Matrix<T>` - матрицы над полем

**Запуск**:
```bash
cd lab_3-5/lab3-rational-numbers
dotnet run
```

## Лабораторная работа №4: Векторы над полем

**Суть**: Реализация класса `Vector<T>` для работы с векторами над любым математическим полем.

**Возможности**:
- Арифметические операции (сложение, вычитание, умножение на скаляр)
- Скалярное произведение (dot product)
- Векторное произведение (cross product для 3D)
- Парсинг и генерация случайных векторов
- Работа с любым типом, реализующим `IField<T>`

**Запуск**:
```bash
cd lab_3-5/lab4-vector-demo
dotnet run
```

## Лабораторная работа №5: Системы линейных уравнений

**Суть**: Решение систем линейных уравнений Ax = b над математическими полями.

**Возможности**:
- Решение методом Гаусса с частичным выбором ведущего элемента
- Проверка существования и единственности решения (теорема Руше-Капелли)
- Анализ типа решения:
  - Единственное решение: rank(A) = rank([A|b]) = n
  - Бесконечно много: rank(A) = rank([A|b]) < n
  - Нет решения: rank(A) < rank([A|b])
- Автоматическая верификация решения
- Работа с рациональными и комплексными системами

**Запуск**:
```bash
cd lab_3-5/lab5-linear-systems-demo
dotnet run
```
